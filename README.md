A **modern, responsive company website** built with **Node.js, Express.js, EJS, and CSS**.
This project demonstrates server-side rendering using **EJS templates** and a clean, responsive UI suitable for all devices.

---

## 📌 Overview

This project is designed as a **dynamic company website** where the backend handles routing and page rendering while the frontend focuses on responsive design and user experience.

It can be used as a **starter template for business websites, portfolios, or startup landing pages**.

✔️ Fully **Responsive Design**

# ⚙️ Installation

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/company-website.git
```

### 2️⃣ Navigate to the project folder

```bash
cd company-website
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the server

```bash
node app.js
```

or with **nodemon**

```bash
nodemon app.js
```

# ▶️ Usage

After running the server, open your browser and visit:
